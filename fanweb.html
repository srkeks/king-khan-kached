The Residue

Scraping the web for SRK. Some images are degraded, compressed past recognition. Some text is dead, still readable but torn out of the page that gave it meaning. Some links are only a trace, pointing into a site that no longer loads. And some fragments are ghosts — analytics beacons, blank links, the residue the scrape caught but no one was meant to keep. This grid is the index of the whole scrape; the fragments that carry weight have a ▾ describe button that opens a full record in place — Dublin Core metadata and a separate metadata-of-loss block naming what cannot be recovered.

✦ a scrape is not a copy but a tearing-out. what you see here lost its page to get here. ✦
